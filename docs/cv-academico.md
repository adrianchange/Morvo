## Descripción formal

Desarrollo de una aplicación web interactiva destinada a la presentación institucional de la producción teatral **MORVO** ante salas, festivales y entidades de programación cultural. El sistema integra un dossier navegable de ocho módulos informativos (portada, material gráfico, sinopsis, fichas de interpretación, dirección y teaser audiovisual) con un motor de identidad visual parametrizable que contempla once variantes cromáticas intercambiables.

La arquitectura software se fundamenta en una Single Page Application (SPA) construida con React y TypeScript, compilada mediante Vite y desplegada como artefacto estático. La separación entre capa de contenido y capa de identidad visual — implementada mediante un sistema de tokens temáticos — permite reutilizar la estructura del dossier para distintos contextos de difusión: presentación formal a programadores escénicos y piezas gráficas derivadas para comunicación en redes sociales.

El elemento central de la interfaz es el logotipo animado **MORVO**, cuyo diseño tipográfico remite a la serie *Killing Eve*. El carácter «V» se ha vectorizado a partir de las métricas originales de la fuente digital Killing Eve, habilitando animaciones de goteo específicas por variante cromática, con sincronización temporal entre movimiento gráfico y aparición de créditos textuales.

## Competencias técnicas demostradas

- Desarrollo frontend con React y TypeScript en entorno de producción.  
- Diseño e implementación de sistemas de diseño multi-tema (design tokens).  
- Animación vectorial SVG con biblioteca Motion y keyframes temporales.  
- Tipografía digital: extracción de métricas de fuente, composición tipográfica responsive.  
- Diseño de interacción para presentación escénica (navegación por teclado, gestos táctiles, viewport 16:9).  
- Optimización de build estático y accesibilidad en dispositivos móviles.
