## Resumen corto (Experiencia / Proyecto — ~600 caracteres)

Desarrollé el dossier digital de **MORVO**, una producción de Compañía OBSCENA Teatral: una web interactiva en React + TypeScript donde programadores y festivales recorren sinopsis, elenco y teaser en un formato visual cuidado.

Lo diferencial: un **sistema de 11 paletas de color** intercambiables — una para el dossier institucional, el resto para redes y difusión — y un **logotipo MORVO animado** con el glifo V vectorizado (tipografía Killing Eve) y efectos de goteo únicos por variante.

Stack: React 19 · Vite · Motion · SVG · diseño responsive.

## Publicación sugerida (post)

🎭 Acabo de publicar **MORVO**, el dossier digital interactivo de nuestra producción teatral.

No es un PDF más: es una web navegable donde salas y festivales pueden recorrer sinopsis, elenco, directora y teaser — con un diseño pensado para impresionar en pantalla.

Lo que más me ha costado (y más me ha gustado): el título **MORVO** animado. La V está vectorizada desde la tipografía de *Killing Eve*, con un sistema de goteo distinto para cada una de las 11 paletas de color del proyecto.

Una paleta para el dossier. Diez más para redes y campaña gráfica. Misma estructura, identidad visual distinta.

Stack: React 19 · TypeScript · Vite · Motion
